Vim - Basics

Vim is a text editor for Unix/Linux that can run in a terminal. It comes preinstalled on most, if not all, versions of Unix or Linux.

Command Description

vim <example_file>

From the Linux terminal, opens a file in a Vim editor

Press i

Enable editing/insert mode

Press Ctrl + c or Esc

Exit editing/insert mode

:wq

Save and quit the editor

:w

Save without quitting the editor

:q

Quit the editor without saving

:q!

Quit the editor without saving (ignores unsaved changes)

set number

Shows line numbers in the editor

:<line_number>

Jumps to a specific line number

:$

Jumps to the last line
dd

Deletes the current line.
:<start_position>,<end_position>d

Delete lines starting from the specified start_position and ending at the specified end_position.

/<target_string>
?<target_string>

/<target_string> searches for matches forward from the current position while ?<target_string> searches backwards from the current position. Navigate through the results using the n to move forwards and N to move backwards.
