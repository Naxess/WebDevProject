Django - Admin Interface

Creating a Superuser Account

To use the Admin interface, we need to create a superuser account. This can be done using with this command: 

python manage.py createsuperuser

Registering the Models

In order to use the Admin interface to interact with the models, we need to first register the models to the app's admin.py file. In admin.py, the first thing that needs to be done is to import the models from the app. With that done, we need to call the admin.site.register method once for each imported model. Each call to the register function should be supplied with one of the imported models.
Example: 

from django.contrib import admin
from appName.models import ExampleModelA, ExampleModelB

admin.site.register(ExampleModelA)
admin.site.register(ExampleModelB)

Interacting with the Models

Access to the (default) Admin interface is through this URL (http://127.0.0.1:8000/admin/). Sign in using a superuser account. After signing in, the home page of the Admin page should display all created (and registered) tables (models) in the Site administration section (ignore the AUTHENTICATION AND AUTHORIZATION section, that one is created by Django by default).

On the right side of each table, there are two buttons: Add and Change. Clicking on Add will open a form that accepts values and when submitted will create a row in the table populated with the supplied values. Clicking Change will open a form that allows us to change the values on an existing row in the table.
