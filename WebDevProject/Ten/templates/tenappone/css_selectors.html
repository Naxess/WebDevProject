CSS - Selectors

In CSS, selectors are patterns used to select desired elements from a HTML webpage. Once selected, changes and other effects can be applied to the elements.

Syntax:
    <selector> {
        <propertyName>: <propertyValue>;
        <propertyName2>: <propertyValue2>;
        ...
    }
Example usage:
    body {
        margin: 10px;
        padding: 10px;
        background-color: gray;
    }

Types of Selectors
Selector Functionality Syntax/Example Usages
Universal Selector Using an asterisk ( * ) as the selector will select everything in the document. 
* { ... }
<body>
    <h1>Selected</h1>
    <h2>Selected</h2>
    <h3>Selected</h3>
</body>

// Selects every element on the page
* { ... }
Element (AKA 'Tag name' or Type) Selector Selects all instances of the specified tag/element name (i.e. html elements such as body, h1, div, p, etc..). To select multiple different tags, use a comma as a delimiter (ex. h1, h2, h3 { ... }). 
elementName { ... }
elementName, element2Name, element3Name { ... }

<body>
    <h1>Selected</h1>
    <h2>Selected</h2>
    <h3>Selected</h3>
</body>

// Selects the 'body' element
body { ... }
// Selects all 'h1', 'h2', and 'h3' elements
h1, h2, h3 { ... }
ID Selector Selects the element on the page with the specified ID. IDs must start with a hashtag, like so: #exampleId, #JohnDoe, #idsMustBeUnique, etc.. 
#id { ... }
<body>
    <h1 id="ExampleId">Selected</h1>
</body>

// Selects the 'h1' element with the 'id' attribute of 'ExampleId'
#ExampleId { ... }
Class Selector Selects elements on the page whose class attribute matches the specified value. Classes must start with a dot (period), like so: .exampleClass, .ClassOf2024, .classy 
.className { ... }
<body>
    <h1 class="exampleClass">Selected</h1>
</body>

// Selects the 'h1' element with a 'class' attribute of 'exampleClass'
.exampleClass { ... }
Class-Value Selector V2 Elements can be selected based on the value assigned to the class attribute by specifying the element name and the value assigned to the class attribute, separated by a dot. Example: span.highlight will select <span class="highlight"></span> 
elementName.classValue { ... }
<body>
    <h1 class="highlight">Selected by #1</h1>
    <input type="text" class="form-control">Selected by #2</a>
</body>

// #1. Selects 'h1' tags with a 'class' attribute value of 'highlight'
h1.highlight { ... }

// #2. Selects 'input' tags with a 'class' attribute value of 'form-control'
input.form-control { ... }
Attribute Selector

Selects elements on the page that match the specified element and contain the specified attribute. For attributes, results can be further narrowed by specifying the value. The selection criteria can be expanded to match elements with attribute values that contain the specified value. Additionally, adding the letter i into the square bracket after the specified value allows the search to ignore case-sensitivity.

[attributeName]
Matches any elements that contain the provided attribute
~=
Matches elements whose attribute values contain the specified value in its space-delimited list of values
|=
Matches elements whose attribute value is exactly the specified value or begins with the value followed by a hyphen ( - )
^=
Matches elements whose attribute value begins with the specified value
$=
Matches elements whose attribute value ends with the specified value
*=
Matches elements whose attribute value contains the specified value as a substring
i
When added to the square brackets, it allows the search to ignore case-sensitivity 
[attributeName]{ ... }
elementName[attributeName]{ ... }
elementName[attributeName='attributeValue']{ ... }
elementName[attributeName~='attributeValue']{ ... }
elementName[attributeName|='attributeValue']{ ... }
elementName[attributeName^='attributeValue']{ ... }
elementName[attributeName$='attributeValue']{ ... }
elementName[attributeName*='attributeValue']{ ... }
elementName[attributeName='attributeValue' i]{ ... }

<body>
    <a>This won't be selected - no 'href' attribute.</a>
    <a href="#">This will be selected by #1</a>
    <a href="https://www.google.com/">This will be selected by #1 and #2</a>
    <img src="profile_pic.png" alt="This will be selected by #3 and #4">
    <video src="audition.mp4">This will be selected by #4</video>
</body>

// #1. Matches the 2nd and 3rd 'a' tags
a[href] { ... }

// #2. Matches the 3rd 'a' tag
a[href="https://www.google.com/"] { ... }

// #3. Matches the 'img' tag
img[src="profile_pic.png"] { ... }

// #4. Matches all elements that contain the 'src' tag, regardless of type
[src] { ... }
Pseudo-class Selector Selects elements on the page that match the provided element and attached pseudo-class. Pseudo-classes can be identified because they start with a single colon.
Examples of pseudo-classes: :active, :blank, :disabled, :enabled, :hover, :optional, :required, :paused, :playing, :stalled, :read-only, :read-write, :focus, :first-child, :last-child) pseudo-elements (which select a certain part of an element rather than the element itself). 
elementName:pseudoClass { ... }
<body>
    <p>This won't be selected</p>
    <p>This element will be selected because it's the last sibling.</p>
</body>

// Selects the last 'p' element that is a descendant of 'body'
body p:last-child { ... }

// Selects the element only when the mouse pointer is hovering over it
a:hover { ... }
Pseudo-element Selector Selects parts of elements in the document that match the provided pseudo-elements. For example, p::first-line will return the first-line of text in a paragraph, but not the paragraph element itself. Pseudo-elements can be identified because they start with a double colon.
Examples of pseudo-elements: ::first-letter, ::first-line, ::placeholder, ::selection, ::before, ::after, ::grammar-error, ::spelling-error 
elementName::pseudoElement { ... }
<body>
    <p>This text (not the element itself) will be selected</p>
</body>

// Selects the first line of text inside of the element
// (instead of the element itself)
p::first-line { ... }

<body>
    <p>This text (not the element itself) will be selected</p>
    <p>This won't selected, because it isn't the first sibling.</p>
</body>

// Usages of pseudo-elements can be combined with pseudo-classes
body p:first-child::first-line { ... }
Descendant-Combinator Selector Combines two or more selectors, each delimited by a single space. The first (leftmost) selector is the parent, with each following selector being a descendant of the previous one. parent descendant descendantOfDescendant { ... } 
<body>
    <em>
        <u>
            <s>This element will be selected.</s>
        </u>
    </em>
</body>

// Selects all 's' elements that are descendants of 'em u'

em u s { ... }
Child-Combinator Selector Combines two selectors, delimited by a greater-than symbol ( > ) that matches only those elements matched by the second selector that are the direct children of elements matched by the first. The right selector must be a direct child of the left selector. 
parentElement > desiredElement { ... }
<body>
    <ul>
        <li>This element will be selected.</li>
        <li>This will also be selected.</li>
        <li>This as well.</li>
    </ul>
</body>

// Selects all 'li' elements that are children of 'ul'
ul > li { ... }
Next-sibling Combinator Selector Combines two selectors, delimited by a plus sign ( + ) that matches only those elements matched by the second selector that are immediately preceded by the first selector. 
precedingSibling + desiredSibling { ... }
<body>
    <h1>Hello</h1>
    <p>This element will be selected.</p>
    <p>This won't because it doesn't appear immediately after.</p>
</body>

// Selects all 'p' elements that are preceded by 'h1'
h1 + p { ... }
Subsequent-sibling Combinator Selector Allows selecting siblings of an element even if they are not directly adjacent. The right selector matches all elements that come anywhere after the left selector even if there are non-target elements in between. 
precedingSibling ~ desiredSiblings { ... }
<body>
    <h1>Hello</h1>
    <p>This element will be selected.</p>
    <img src="#" alt="Won't be selected.">
    <p>This will also be selected.</p>
</body>

// Selects all 'p' elements that appear after 'h1'
h1 ~ p { ... }
