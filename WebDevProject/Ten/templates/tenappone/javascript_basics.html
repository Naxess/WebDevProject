Variables

Declaring and using variables. Variable keywords: let, const, and var

The 'let' keyword is recommended for variable declaration. The variable cannot be redeclared. The value can be updated. Block-scoped. 

let variableName;
variableName = "hello";
variableName;                           // Output: 'hello'

The 'const' keyword is meant for use with constants, variables whose value will not be changeable. The variable cannot be redeclared. Its properties can be updated but key-value pairs cannot be added or removed. Blocked-scoped. 

const variableName2 = 'hamburger';

The 'var' keyword is used for variable declaration. The variable can be redeclared and this is why it is no longer recommended for regular use. Globally-scoped if declared outside of a function - meaning it's accessible throughout the whole window. Function-scoped if declared within a function - meaning it's only accessible within the function. 

var variableName3 = 'french fries';

Printing to the console - using console.log()

The console.log() method can take an optional argument and print that argument to the console. If no argument is supplied, then it only outputs a newline.
There is no return value. 

    console.log(message);

Alert, Confirm, and Prompts

The alert() method, when called, will display an alert box on top of the webpage with a message and an OK button.
There is no return value. 

    alert(message);

The confirm() method, when called, will display a dialog box with a message, an OK button, and a Cancel button.
Returns true if user clicked the OK button, else false. 

    confirm(message);

The prompt() method, when called, will display a dialog box with a message, a text input field, an OK button, and a Cancel button.
Returns the value in the text input field as a string if the user clicked the OK button, else null. 

    prompt(message, placeholderText);

Classes

 
class ClassName{
    public_variable = 321;
    #private_variable = 123;             // Class fields can be designated as private by using '#'
    static static_variable = 312;

    constructor(first_name, last_name){
        this.first_name = first_name.toUpperCase();
        this.last_name = last_name.toUpperCase();
    }

    get_full_name(){
        return `${this.last_name}, ${this.first_name}`;
    }

    set set_first_name(first_name){
        /*
            The use of 'set' permits only a single parameter. Value setting syntax: obj.set_first_name = 'name';
            Note that it does not use () and the value is set using an equality operator.
        /*
        this.first_name = first_name.toUpperCase();
    }

    set set_last_name(last_name){
        this.last_name = last_name.toUppercase();
    }

    update_full_name(first_name, last_name){
        this.first_name = first_name.toUpperCase();
        this.last_name = last_name.toUpperCase();
    }

    get get_first_name(){
        /*
            The 'get' keyword binds an object property to a function. Usage syntax: obj.get_first_name;
            Note that it does not use () and it called like a variable.
        */
        return this.first_name;
    }

    get get_last_name(){
        return this.last_name;
    }

    // Getting and setting private variables
    get get_private(){
        return this.#private_variable;
    }

    set set_private(new_private){
        this.#private_variable = new_private;
    }

    static static_method(upperLimit){
        return Math.floor(Math.random() * upperLimit);
    }
}

// shift, unshift, push, pop
var hello_arr = ["H", "E", "L", "L", "O"];
console.log("Unmodified array: "+hello_arr);
hello_arr.shift();
console.log("Shift removes the first element in an array: " + hello_arr);
hello_arr.unshift("Y", "O", "J");
console.log("Unshift adds n number of elements to the beginning of an array: " + hello_arr);
hello_arr.pop();
console.log("Pop removes the last element in an array: " + hello_arr);
hello_arr.push("I", "N", "G");
console.log("Push adds n number of elements to the end of an array: " + hello_arr);
