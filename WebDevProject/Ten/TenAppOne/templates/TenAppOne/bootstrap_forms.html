Bootstrap - Forms

Form Control

Input
HTML Code:
<div class="mb-2"> <!--mb-2 is margin-bottom-3. Adds 3 spacing to the margin-bottom. Allowed values: 0-5.-->
    <label for="text_one" class="form-label">Enter desired username:</label>
    <input type="text" class="form-control" id="text_one" placeholder="Placeholder text here...">
</div>

Textarea
HTML Code:
<div class="mb-2"> <!--Sets margin-bottom spacing to 4-->
    <label for="textarea_one" class="form-label">Enter a brief description of yourself:</label>
    <textarea class="form-control" id="textarea_one" rows="3"></textarea>
</div>

Additional Sizes
HTML Code:
<div class="mb-0"> <!--Sets margin-bottom spacing to 0-->
    <label for="email_one" class="form-label">Resized to Small</label>
    <input type="email" class="form-control form-control-sm" id="email_one" placeholder="Small input">
</div>
HTML Code:
<div class="mb-2"> <!--Sets margin-bottom spacing to 0-->
    <label for="number_one" class="form-label">Resized to Large</label>
    <input type="number" class="form-control form-control-lg" id="number_one" placeholder="Large input">
</div>

Disabled/Read-only Attributes
HTML Code:
<input type="text" class="form-control" placeholder="This input is disabled" disabled>
<input type="text" class="form-control" placeholder="This input is set to read-only" readonly>
<input type="text" class="form-control" placeholder="Disabled and read-only" disabled readonly>

File Upload
HTML Code:
<label for="file_one" class="form-label">Input for uploading files</label>
<input type="file" class="form-control" id="file_one" multiple>

Color-Picker
HTML Code:

<input type="color" class="form-control form-control-color" id="color_one">

Datalist
HTML Code:
<label for="datalist_one" class="form-label">Datalist is like a Select but with more style options</label>
<input class="form-control" list="datalist_options_one" id="datalist_one" placeholder="Click here, pick one">
<datalist id="datalist_options_one">

    <option value="Three"></option>
    <option value="Four"></option>
</datalist>

Ranges

 1

<output>1</output>

Layout

Utilizing Bootstrap's grid system: rows (row) and columns (col)

Basic Grid

Using "col" without specifying sizing (e.g. col-) will result in equal-width columns that span the full width of container.

Column One
Column Two
Column Three
HTML Code:
<div class="row">
    <div class="col">Column One</div>
    <div class="col">Column Two</div>
    <div class="col">Column Three</div>
</div>

Grid with Column-Sizing

The grid system has 12 slots for columns. These slots are divided up amongst the supplied columns evenly if the columns do not explicitly request x number of slots. Each column can request a single slot or up to a maximum of 12 slots.
