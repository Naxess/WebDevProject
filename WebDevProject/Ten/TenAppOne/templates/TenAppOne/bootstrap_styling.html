Bootstrap - Styling

Spacing

 https://getbootstrap.com/docs/5.3/utilities/spacing/

The spacing utility allows developers to set the margin and padding of elements from an HTML elements' class attribute. These properties can be further modified to specify the spacing level, desired side to edit, the breakpoint level, etc..

Syntax formats: 

<property><sides>-<size>
<property><sides><breakpoint>-<size>
Properties
Property Description
m

Shorthand for margin, which is responsible for setting the spacing directly outside of an element's border.

p

Shorthand for padding, which is responsible for setting the spacing between an element's border and its text content.

Sides
Property Description
t

Shorthand for classes that set margin-top or padding-top.

b

Shorthand for classes that set margin-bottom or padding-bottom.

s

Shorthand for classes that set margin-left or padding-left.

e

Shorthand for classes that set margin-right or padding-right.

x

Shorthand for classes that set both *-left and *-right.

y

Shorthand for classes that set both *-top and *-bottom.

(blank)

Shorthand for classes that set margin or padding on all sides of the element.

Breakpoints
Property Description
xs

Less than 576px.

sm

Greater than or equal to 576px.

md

Greater than or equal to 768px.

lg

Greater than or equal to 992.

xl

Greater than or equal to 1200px.

xxl

Greater than or equal to 1400px.

Size
Property Description
0

Sets margin or padding to 0.

1

Sets margin or padding to $spacer * .25.

2

Sets margin or padding to $spacer * .5.

3

Sets margin or padding to $spacer.

4

Sets margin or padding to $spacer * 1.5.

5

Sets margin or padding to $spacer * 3.

auto

Sets margin to auto.

Horizontal Centering

You can center an element on the page by using mx-auto. Example:

Centered Element
HTML Code:
<div class="mx-auto bg-secondary text-light px-1" style="width:200px;">
    Centered Element
</div>

Gutters

 https://getbootstrap.com/docs/5.3/forms/layout/#gutters

Utility to edit the padding between columns of content in Bootstrap's grid system. Gutters are the gaps between column content, created by horizontal padding (padding-right and padding-left). Negative margin is used to offset the start and end of each row to align content. Horizontal/vertical, and all other gutters can be adjusted.

Text Utilities

The text and color utilities are used to change the text alignment, transform, style, weight, line-height, decoration, and color:

Abbreviations

The <abbr> element applies an underline effect to the enclosed text. If the title attribute is supplied, then it will be displayed when the mouse pointer is hovered over the element.

Hover over me!

Hover over me!

HTML Code:
<p><abbr title="Hello!">Hover over me!</abbr></p>

Blockquotes

A well-known quote, contained in a blockquote element.

HTML Code:
<blockquote class="blockquote">
  <p>A well-known quote, contained in a blockquote element.</p>
</blockquote>

Figure, Figcaption, and Cite

Blockquotes are typically used in conjunction with <figure> and <figcaption elements>, specifically, enclosed within the former and preceding the latter. <cite> is typically within the <figcaption>, enclosed around the source title.

HTML Code:
<figure>
  <blockquote class="blockquote">
    <p>A well-known quote, contained in a blockquote element.</p>
  </blockquote>
  <figcaption class="blockquote-footer">
    Someone famous in <cite title="Source Title">Source Title</cite>
  </figcaption>
</figure>

Unstyled Lists

Setting the class attribute to list-unstyled removes the bullet point from (direct) list-items. Nested lists are unaffected.

  • This is a list.
  • It appears completely unstyled.
  • Structurally, it's still a list.
  • However, this style only applies to immediate child elements.
  • Nested lists:
    • are unaffected by this style
    • will still show a bullet
    • and have appropriate left margin
  • This may still come in handy in some situations.
HTML Code:
<ul class="list-unstyled">
    <li>This is a list.</li>
    <li>It appears completely unstyled.</li>
    <li>Structurally, it's still a list.</li>
    <li>However, this style only applies to immediate child elements.</li>
    <li>Nested lists:

            <li>are unaffected by this style</li>
            <li>will still show a bullet</li>

</ul>
