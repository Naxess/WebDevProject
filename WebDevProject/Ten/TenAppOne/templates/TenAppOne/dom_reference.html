HTMLCollection & NodeList: Similarities and Differences
  • HTMLCollections are similar to NodeLists in that they are both array-like collections (lists) of nodes (elements) from a document.
  • HTMLCollection items can be accessed by their name, id, or index number. NodeList items can only be accessed by their index number.
  • An HTMLCollection is a collection of document elements while NodeList is a collection of document nodes (element/attribute/text nodes).
  • Also, HTMLCollection is always a live collection (meaning that it will change along with any additions/subtractions). A NodeList is usually a static collection (it will be unaffected by any additions/subtractions).

HTML DOM: Style Object - Properties

The style attribute is available to HTMLElements. It returns the inline style of an element in the form of a live CSSStyleDeclaration object that contains a list of all style properties for that element with values assigned only for the attributes that are defined in the element's inline style attribute. Inline styles can be applied by assigning a string directly to the style property. Use style in this manner will completely overwrite all inline styles on the element.
