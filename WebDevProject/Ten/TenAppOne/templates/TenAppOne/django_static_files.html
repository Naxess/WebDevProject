Django - Static Files

Static files typically refer to files like media (images/videos/audio), Javascript, CSS, etc., that are used by websites for whatever reason.

Static Files Initial Setup

  • Before we can start using static files, we need to create a directory in the root of the project directory, named 'static'. Subdirectories are optional but it's probably recommended to create at least three - one for Javascript files, one for CSS files, and another for media files.

  • The next step is to register the static files directory. Open the settings.py file, and run the following steps:

    • Import the os module (if it hasn't been imported already). (Alternatively, can import just the path function from the os module. If you do this, adjust the value in the next step accordingly.) 

      import os

    • Check to see if there is a variable named STATIC_URL - it may be located near the bottom of the settings.py file. It should have a string value of "static/". If the variable doesn't exist or the value is different, then add/edit the variable to match. 

      STATIC_URL = "static/"

    • Next, locate a variable named BASE_DIR. Directly below that variable, create another two variables:

      • First variable should be named 'STATIC_DIR' with a value set to os.path.join(BASE_DIR, 'static').

      • Then create 'STATICFILES_DIRS' below. Set the value as a list object with one item: STATIC_DIR.

Using Static Files

  • Before an HTML page can start loading files from the static directory, it needs to insert the following tag (preferably near the top, before any attempts to load the files): 

    {{"{"}}% load static %{{"}"}}

  • Loading a file from the static directory is done by using the following template tag syntax: 

    {% verbatim %}{% static 'static_file_path' %}{% endverbatim %}
    Example usage:
    {% verbatim %}
<html>
    {% load static %}
    <head>
        <title>Django - Static Files</title>
        <link rel="stylesheet" href="{% static 'css/django_static_files.css' %}">
    </head>
    <body>
        <h1>Django - Static Files</h1>
        <img src="{% static 'media/images/cat.png' %}" alt="cat.png" alt="">
    </body>
</html>
{% endverbatim %}
